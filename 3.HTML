It is a cross platform or object-oriented scripting language that is responsible for the operation of web pages. For example, using JavaScript, you can issue commands such as "Click the button and show me the date below."

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page.
